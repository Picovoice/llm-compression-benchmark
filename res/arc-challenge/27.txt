Students heated three objects to different temperatures during a classroom demonstration. Each object emitted light of a different color as shown below. Object 1: blue light Object 2: red light Object 3: orange light Which list presents the objects in order from highest to lowest temperature? (A) Object 1, Object 2, Object 3 (B) Object 1, Object 3, Object 2 (C) Object 2, Object 1, Object 3 (D) Object 2, Object 3, Object 1
Answer: B
